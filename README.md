## not thirsty anymoreee web application

not thirsty anymoree is a dynamic web application. It utilises basic dynamic functionality by iterpreting data, input by the user via the use of html forms and inputs, in order to render relevant data to the user on the webpages.

The application utilises the Express web framework and Embedded JavaScript (EJS) via Node.js.
It is run on a server hosted on a virtual machine, and kept live at all times using the command line interface tool Forever via also via Node.js.
